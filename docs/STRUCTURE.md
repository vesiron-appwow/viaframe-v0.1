# VIAFRAME Folder Structure

This document defines the purpose of each top-level folder in VIAFRAME.

## /core
Reusable chassis components shared by all Linx apps.
Includes layout, navigation, utilities, and base logic.

## /apps
Individual demo or product apps built on top of VIAFRAME.
Each app is self-contained and may be deployed independently.

## /docs
Human-readable documentation.
Used to explain architecture, decisions, and structure.

## /config
Shared configuration files, feature flags, and environment templates.
