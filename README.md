# VIAFRAME v0.1

VIAFRAME is a static, data-driven Instant App chassis built with Astro.

## Status
- v0.1 reference locked
- Static routing
- JSON-driven templates
- No backend
- Cloudflare Pages ready

## Purpose
VIAFRAME provides a clean foundation for building and deploying
single-purpose Instant Apps under the AppWow platform.

Part of the VESIRON / AppWow initiative.
